---
title: Factorial of Number Using Recursion
description: The factorial of a number is the product of all the numbers from 1 to that number.
logo: /static/images/algorithm/algorithm.png
categories: ['Recursion']
---

```js
function fact(n) {
  if (n < 0) throw new Error('Number must be positive');

  if (n === 0) return 1;

  return n * fact(n - 1);
}

console.log(fact(5));
```

Time Complexity O(n)/Space Complexity O(n)

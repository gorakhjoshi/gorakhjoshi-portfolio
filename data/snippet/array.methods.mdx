## Array Methods

1. **at**: Returns the element at a specified index in the array.
2. **concat**: Joins two or more arrays and returns a new array.
3. **copyWithin**: Copy a portion of an array to another location within the same array without changing its size.
4. **entries**: Returns a new array iterator object containing key/value pairs for each index in the array.
5. **every**: Checks if all elements in the array satisfy a provided condition.
6. **fill**: Fills elements of an array with a static value from a start index to an end index.
7. **filter**: Creates a new array containing elements that pass the specified condition.
8. **find**: Returns the value of the first element in the array that satisfies a provided condition.
9. **findIndex**: Returns the index of the first element in the array that satisfies a provided condition.
10. **findLast**: Find the last element in an array that satisfies a given condition.
11. **findLastIndex**: Find the index of the last element in an array that satisfies a given condition.
12. **flat**: Creates a new array with all sub-array elements concatenated into it recursively up to a specified depth.
13. **flatMap**: Maps each element using a mapping function and flattens the result into a new array.
14. **forEach**: Executes a provided function once for each array element.
15. **includes**: Checks if an array contains a specific element.
16. **indexOf**: Returns the first index at which a specified element can be found in the array.
17. **join**: Joins all elements of an array into a string.
18. **keys**: Return a new Array Iterator object that contains the keys for each index in the array.
19. **lastIndexOf**: Returns the last index at which a specified element can be found in the array.
20. **map**: Executes a provided function once for each array element and creates a new array with the results.
21. **pop**: Removes the last element from an array and returns that element.
22. **push**: Adds one or more elements to the end of an array and returns the new length.
23. **reduce**: Reduces the array to a single value by applying a function to each element.
24. **reduceRight**: Similar to `reduce`, but processes the array from right to left.
25. **reverse**: Reverses the order of elements in an array.
26. **shift**: Removes the first element from an array and returns that removed element.
27. **slice**: Returns a shallow copy of a portion of an array as a new array.
28. **some**: Checks if at least one element in the array satisfies a provided condition.
29. **sort**: Sorts the elements of an array in place.
30. **splice**: Adds or removes elements from an array at a specified index.
31. **toLocaleString**: Returns a Date object as a string, using locale settings.
32. **toReversed**: Create a new array with the elements of the original array in reversed order.
33. **toSorted**: Create a new array with the elements of the original array sorted.
34. **toSpliced**: Create a new array with some elements added, removed, or replaced without modifying the original array.
35. **toString**: Returns a string representing the array and its elements.
36. **unshift**: Adds one or more elements to the beginning of an array and returns the new length.
37. **values**: Return a new Array Iterator object that contains the values for each index in the array.
38. **with**: Returns a copy of the array it's called on with the index set to the new value you provide.

## String Methods

1. **length**: Returns the number of characters in a string.
2. **concat**: Combines two or more strings and returns a new string.
3. **substring**: Extracts a portion of a string based on start and end indexes.
4. **indexOf**: Returns the index of the first occurrence of a specified substring within a string.
5. **replace**: Replaces specified substrings within a string with new values.
6. **toUpperCase**: Converts all characters in a string to uppercase.
7. **toLowerCase**: Converts all characters in a string to lowercase.
8. **startsWith**: Checks if a string starts with a specific substring.
9. **endsWith**: Checks if a string ends with a specific substring.
10. **trim**: Removes whitespace from the beginning and end of a string.
11. **split**: Divides a string into an array of substrings based on a specified separator.
12. **charAt**: Retrieves the character at a specific index within a string.
13. **charCodeAt**: Returns the Unicode value of a character at a specified index.
14. **includes**: Checks if a string contains a specific substring.
15. **match**: Determines if a string matches a particular pattern using regular expressions.
16. **repeat**: Repeats a string a specified number of times.
17. **lastIndexOf**: Finds the index of the last occurrence of a specified substring within a string.
18. **slice**: Extracts a portion of a string based on start and end indexes.
19. **isEmpty**: Checks if a string is empty.
20. **isNaN**: Determines if a string represents a valid number.

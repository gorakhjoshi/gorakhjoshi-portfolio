From the lists provided, here are some widely used and popular methods for JavaScript arrays and strings:

### Array Methods

1. `concat()`: Combines arrays by concatenating them into a new array.
2. `filter()`: Creates a new array with elements that pass a test defined by a callback function.
3. `forEach()`: Calls a provided function once for each element in an array.
4. `map()`: Creates a new array populated with the results of calling a provided function on every element in the calling array.
5. `push()`: Adds one or more elements to the end of an array and returns the new length of the array.
6. `pop()`: Removes the last element from an array and returns that element.
7. `reduce()`: Reduces the array to a single value, executing a provided function for each value of the array (from left to right).
8. `shift()`: Removes the first element from an array and returns that removed element.
9. `unshift()`: Adds one or more elements to the beginning of an array and returns the new length of the array.
10. `slice()`: Returns a shallow copy of a portion of an array into a new array object.
11. `some()`: Tests whether at least one element in the array passes the test implemented by the provided function.
12. `sort()`: Sorts the elements of an array in place and returns the array.
13. `splice()`: Changes the contents of an array by removing or replacing existing elements and/or adding new elements.
14. `indexOf()`: Returns the first index at which a given element can be found in the array, or -1 if it is not present.
15. `includes()`: Determines whether an array includes a certain value among its entries, returning true or false as appropriate.

### String Methods

1. `charAt()`: Returns the character at the specified index (position) in a string.
2. `concat()`: Combines the text of two or more strings and returns a new string.
3. `includes()`: Checks if a string contains the specified substring.
4. `indexOf()`: Returns the position of the first occurrence of a specified value in a string.
5. `match()`: Searches a string for a match against a regular expression and returns the matches.
6. `replace()`: Searches a string for a specified value or regular expression and returns a new string where the specified values are replaced.
7. `slice()`: Extracts a section of a string and returns it as a new string.
8. `split()`: Splits a string into an array of substrings based on a specified separator.
9. `substring()`: Extracts the characters from a string, between two specified indices, and returns the new sub-string.
10. `toLowerCase()`: Converts a string to lowercase letters.
11. `toUpperCase()`: Converts a string to uppercase letters.
12. `trim()`: Removes whitespace from both ends of a string.
13. `startsWith()`: Checks whether a string begins with specified characters.
14. `endsWith()`: Checks whether a string ends with specified characters.
15. `repeat()`: Returns a new string with a specified number of copies of the string it was called on.

These methods are commonly used in JavaScript development for manipulating arrays and strings effectively.

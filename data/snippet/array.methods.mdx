## Array Methods

1. **map**: Executes a provided function once for each array element and creates a new array with the results.
2. **filter**: Creates a new array containing elements that pass the specified condition.
3. **reduce**: Reduces the array to a single value by applying a function to each element.
4. **reduceRight**: Similar to `reduce`, but processes the array from right to left.
5. **push**: Adds one or more elements to the end of an array and returns the new length.
6. **pop**: Removes the last element from an array and returns that element.
7. **shift**: Removes the first element from an array and returns that removed element.
8. **unshift**: Adds one or more elements to the beginning of an array and returns the new length.
9. **splice**: Adds or removes elements from an array at a specified index.
10. **slice**: Returns a shallow copy of a portion of an array as a new array.
11. **concat**: Joins two or more arrays and returns a new array.
12. **some**: Checks if at least one element in the array satisfies a provided condition.
13. **every**: Checks if all elements in the array satisfy a provided condition.
14. **fill**: Fills elements of an array with a static value from a start index to an end index.
15. **reverse**: Reverses the order of elements in an array.
16. **includes**: Checks if an array contains a specific element.
17. **at**: Returns the element at a specified index in the array.
18. **find**: Returns the value of the first element in the array that satisfies a provided condition.
19. **forEach**: Executes a provided function once for each array element.
20. **flat**: Creates a new array with all sub-array elements concatenated into it recursively up to a specified depth.
21. **flatMap**: Maps each element using a mapping function and flattens the result into a new array.
22. **indexOf**: Returns the first index at which a specified element can be found in the array.
23. **findIndex**: Returns the index of the first element in the array that satisfies a provided condition.
24. **lastIndexOf**: Returns the last index at which a specified element can be found in the array.
25. **join**: Joins all elements of an array into a string.
26. **sort**: Sorts the elements of an array in place.
27. **toString**: Returns a string representing the array and its elements.
28. **from**: Creates a new array from an array-like or iterable object.
29. **entries**: Returns a new array iterator object containing key/value pairs for each index in the array.
30. **isArray**: Checks if a value is an array.

## String Methods

1. **length**: Returns the number of characters in a string.
2. **concat**: Combines two or more strings and returns a new string.
3. **substring**: Extracts a portion of a string based on start and end indexes.
4. **indexOf**: Returns the index of the first occurrence of a specified substring within a string.
5. **replace**: Replaces specified substrings within a string with new values.
6. **toUpperCase**: Converts all characters in a string to uppercase.
7. **toLowerCase**: Converts all characters in a string to lowercase.
8. **startsWith**: Checks if a string starts with a specific substring.
9. **endsWith**: Checks if a string ends with a specific substring.
10. **trim**: Removes whitespace from the beginning and end of a string.
11. **split**: Divides a string into an array of substrings based on a specified separator.
12. **charAt**: Retrieves the character at a specific index within a string.
13. **charCodeAt**: Returns the Unicode value of a character at a specified index.
14. **includes**: Checks if a string contains a specific substring.
15. **match**: Determines if a string matches a particular pattern using regular expressions.
16. **repeat**: Repeats a string a specified number of times.
17. **lastIndexOf**: Finds the index of the last occurrence of a specified substring within a string.
18. **slice**: Extracts a portion of a string based on start and end indexes.
19. **isEmpty**: Checks if a string is empty.
20. **isNaN**: Determines if a string represents a valid number.

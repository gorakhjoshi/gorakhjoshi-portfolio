---
title: NPM - Set version to latest
description: How to set a published package version to latest
logo: /static/logos/bash.png
categories: ['npm', 'bash']
---

```bash
npm dist-tag add <package>@<version> <tag>
```

An example of this would be:

```bash
npm dist-tag add @chakra-ui/icons@2.0.17 latest
```
